# 多模态创作能力（Multimodal Creativity）

很多人第一次被 AI 震撼，不是因为“它会写字”，而是因为**它能画图、剪视频、做海报**。这就是多模态（Multimodal）的魅力。

直觉上，多模态意味着：**你不只是在“说话”，而是在“给材料”**。材料越具体，输出越稳定。

**短句总结：** **给素材，比给想象更可靠。**

## 视觉创作：图像生成的“配方”

你有没有想过，为什么同样一句话，生成出来的图差别很大？因为图像生成更像在“配方里做菜”。

一个实用的配方通常包含：

- **主体（Subject）**：要画什么？
- **风格（Style）**：写实、插画、像素风？
- **构图（Composition）**：近景、俯拍、三分法？
- **约束（Constraint）**：尺寸、色调、避免出现什么？

可以这样写：

```text
主体：一只在书桌上睡觉的猫
风格：温暖的手绘插画
构图：俯拍，画面偏左
约束：主色调为浅橙，不要出现文字
```

**短句总结：** **主体 + 风格 + 构图 + 约束**，能显著提升可控性。

## 动态叙事：从画面到故事

视频生成不是“把图动起来”，而是“把故事拆成镜头”。你给它的不是一句话，而是一组**镜头指令**。

一个最小可用的分镜清单是这样的：

- **场景**：发生在哪里？
- **人物与动作**：谁在做什么？
- **镜头**：远景/中景/近景？镜头移动吗？

示例：

```text
镜头 1：城市夜景远景，霓虹灯闪烁，轻微推镜。
镜头 2：近景，女生抬头看雨，表情平静。
镜头 3：中景，女生转身离开，雨声变大。
```

**短句总结：** **先镜头，再画面**，叙事会更稳。

## 工作流整合：把文本、图片、视频串起来

真正能提升效率的，不是单点工具，而是**流程（Workflow）**。

一个常见的多模态工作流是：

1. 用文本生成**脚本/大纲**。
2. 用脚本生成**分镜**。
3. 用分镜生成**图片或视频**。
4. 最后进行**剪辑与排版**。

这样做的好处是：每一步都可回退、可修正、可复用。

> 提醒：多模态创作涉及版权与合规，尤其是商业场景，要明确素材来源与使用范围。

**短句总结：** **流程化** 才能让创作稳定可控。

## 小结

- 多模态创作不是“魔法”，而是**给足素材**。
- 图像生成需要“配方”，视频生成需要“分镜”。
- 工作流让创作从偶然变成可复现。

## 练习

1. 写一个图像生成配方，包含主体/风格/构图/约束。
2. 把一个 30 秒短视频拆成 3 个镜头，并写出镜头描述。
3. 画出你自己的“文本 → 分镜 → 成品”的小流程图。
